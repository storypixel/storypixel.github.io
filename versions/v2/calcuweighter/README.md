# Calcuweighter Landing Page

Marketing website for the Calcuweighter iOS app.

## Files

- `index.html` - Main landing page with hero, features, screenshots, and CTA
- `support.html` - Support and FAQ page
- `images/` - App screenshots from the App Store

## Design

- **Color scheme**: Teal accent (#349e9e) matching the site theme, wood grain background (#f0ebe5)
- **Typography**: SF Pro Display/Text (Apple system fonts)
- **Layout**: Mobile-first responsive design
- **Sections**:
  - Sticky header with CTA
  - Hero with key value props (Photo Magic, Voice Input, Zero Mental Math)
  - Screenshot carousel
  - Feature grid (6 features)
  - How It Works (3 steps)
  - Final CTA section
  - Footer with links

## Key Messaging

**Hero**: "Stop Doing Math at the Gym"
- Emphasizes the three easy input methods: photo scanning, voice input, tap grid
- Zero friction, instant results

**Features**:
- PR History & Charts
- Equipment Customization
- Smart Plate Loading
- Clean Retro Aesthetic
- No BS (no ads, no account, no tracking)
- Works Offline
