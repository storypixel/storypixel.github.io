OpenBB Platform

Installed & ran

Open-source Bloomberg-alternative. One Pythonic API wraps 30+ data providers — yfinance, FRED, SEC, BLS, CFTC, Tiingo, Alpha Vantage, Intrinio, Benzinga, Federal Reserve. Free.

Install

uv venv .venv-openbb --python 3.13
source .venv-openbb/bin/activate
uv pip install openbb

Live test (2026-05-09)

>>> obb.equity.price.historical('NVDA',
        start_date='2026-05-05',
        end_date='2026-05-08',
        provider='yfinance')
2026-05-05  close= 196.50
2026-05-06  close= 207.83
2026-05-07  close= 211.50

>>> obb.news.company('NVDA', limit=3)
- The Nasdaq's top winners are now running
  hotter than in 2000: Chart of the Day
- The Memory Shortage Is Coming for Big Tech's
  Bottom Line. This Company Will Hold Up Better
- Prediction: The Helium Crunch Will Accelerate
  the Reshoring of AI...

Verdict: the killer card. One pip install gets us prices + news + econ data + filings + insider trades + crypto + commodities — all 30+ providers behind a single Python interface. This becomes DIY Daily's pulp-aggregator backbone. Add Quint's keys for premium providers (Benzinga, Intrinio) and the same code switches sources. Open-source, no lock-in, MIT.

openbb.co github Free · paid Pro tier for cloud features

Installed & ran

TradingAgents

OSS cloned

Multi-agent LLM trading framework from Tauric Research. Academic paper (arXiv 2412.20138). v0.2.4 just shipped April 2026 — structured-output agents, LangGraph checkpoints, persistent decision log.

Cloned

git clone https://github.com/TauricResearch/TradingAgents.git

Architecture

tradingagents/
├── agents/
│   ├── analysts/      # market, fundamentals, news, social
│   ├── researchers/   # bull, bear
│   ├── managers/      # research manager, portfolio manager
│   ├── risk_mgmt/
│   └── trader/
├── dataflows/
├── graph/             # LangGraph workflow
└── llm_clients/       # Claude, GPT-5.x,
                       # Gemini 3.x, Grok, DeepSeek,
                       # Qwen, GLM, OpenRouter

Verdict: different shape — they're trying to place trades, not write a newsletter — but the multi-agent pattern (analysts → researchers → managers → trader, with bull/bear pairs) maps cleanly to DIY Daily's "draft → contrarian critique → editor sign-off" loop. Steal the bull/bear adversarial pair for the L3 paragraph: one agent argues Quint's bull case, one argues bear, the synthesis becomes the editorial.

github paper Free · Apache-style license

Tiingo MCP

Installed & ran

Community MCP server wrapping Tiingo's API. Stocks, forex, crypto, news, fundamentals, corporate actions. 17 tools, 4 resources, 5 prompts. Free tier covers daily quotes + news.

Install

uvx tiingo-mcp
# add to Claude Code MCP config:
# "tiingo": { "command": "uvx",
#             "args": ["tiingo-mcp"],
#             "env": { "TIINGO_API_KEY": "..." }}

Tools available

get_stock_metadata()
get_stock_prices()       # OHLCV historical
get_realtime_price()     # IEX top-of-book
get_intraday_prices()    # 1m–1h intervals
get_forex_quote()
get_forex_prices()
get_crypto_*
get_news()
get_fundamentals()
get_corporate_actions()

Verdict: good split-tool MCP — agents query exactly the slice they need rather than scrolling through a giant doc. Tiingo's free tier is the cleanest US-equity history we tested (split & dividend adjusted, error-checked). Pair with Alpha Vantage for redundancy: AV for current quote, Tiingo for clean history, news from either.

github tiingo.com Free tier · $10–$50/mo for premium

Endpoints

Bloomberg AskB

Gated / doc-only

"Biggest rethink of the Terminal in Bloomberg history" per CTO Shawn Edwards (Fortune, April 2026). Natural-language agent that builds investment screens, drafts research reports with sophisticated financial modeling, and produces bull/bear cases on the fly.

Closest comp to a Jarvis-for-finance

Hooks the entire Bloomberg data corpus + the journalist-grade news flow underneath. Encodes domain experts' "tacit knowledge" via human-graded evals (bond covenant experts, equity analysts, market structure wonks, BBG journalists). Pull-based interface — "ask me," not "anticipate for me."

Why we can't use it directly

Terminal-locked. ~$24k/seat/yr. No public API. AskB cannot be embedded.

Verdict: the gold standard for institutional. Pattern to crib: the bull/bear research-report synthesis is exactly DIY Daily's L3-paragraph mechanic, scaled up. AskB's evals approach (humans grade agent outputs against expert benchmarks) is what Quint's voice-corpus + edit-history loop becomes long-term.

Fortune coverage $24k/seat/yr · Terminal-locked

Rogo

Gated / doc-only

Late-stage finance-pro chat agent. $2B valn, $160M Series D in April 2026. Built by ex-IB junior bankers around a kitchen table in 2021. Connects internal docs + external data via integrations.

2026 traction signals

GPT 5.5 + Opus 4.7 in product · EU AI Act compliant · ISO/IEC 42001 certified · Fitch Solutions partnership · Acquired Plux (European market expansion) and Offset (security) · Goldman Sachs & major IBs as customers.

Why we can't use it directly

Enterprise-only sales motion. No self-serve. Demo-request gated.

Verdict: closest "Cursor for finance pros" — chat over your firm's internal docs + market data. Pattern to crib: their data-onboarding flow (firms upload internal docs and licensed feeds, Rogo gates by role). Our equivalent: Quint uploads his archive (DIY Money transcripts, Joule blog, X feed) once; AI works against it forever.

Hebbia

Gated / doc-only

"Matrix" UI — agentic doc-synthesis at portfolio scale. Sentence-level citations. Customers run portfolio-wide agent passes for buried risks (unrestricted-subsidiary "drops," subordination clauses, etc.).

What's distinct

Reasons over thousands of proprietary documents at once. Workflow automation across analysis + content creation. Multi-document reasoning (PDFs, slides, spreadsheets, research reports) in a single query.

Why we can't use it directly

Enterprise-only. Custom deployments for hedge funds + private credit shops.

Verdict: Matrix UI is the right surface for the "what do all 14 sources I trust say about today's jobs print" question. Pattern to crib: the matrix layout (rows = entities, columns = questions, cells = AI answers + cite). For DIY Daily, that's rows = headlines, columns = (relevance to holdings? matches a theory? worth commentary?), cells = AI verdict + source link.

hebbia.com Enterprise-only · pricing on request

Moody's MCP

Use cases

Compliance, credit analysis, business development, KYC. Sits inside Claude's interface — Moody's tools and UI surface natively, not just data.

Why we can't use it directly

The MCP-app-not-just-connector framing matters: it embeds the provider's UI inside the agent.

Guidepoint MCP

Gated / doc-only

Anthropic launch partner (May 6, 2026). 100,000+ expert-interview transcripts available inside Claude. Every insight attributed and linked back to its source in Guidepoint360.

What it unlocks

Primary research at agent speed — "what are tier-1 auto suppliers saying about Q3 demand?" answered with sourced expert quotes, not LLM speculation.

Why we can't use it directly

Guidepoint client subscription required (typical buy-side / consulting tier). Standalone access requested via sales@guidepoint.com.

Verdict: the closest commercial answer to "give Quint primary research that's not LLM-default." For DIY Daily v1 we can't subscribe, but the pattern is the gold standard: signal grounded in attributable expert quotes, not a model's training-data echo. Our open-source equivalent: Quint's own podcast transcripts + his CNBC clips become the primary-source corpus.

What this maps to for DIY Daily

The four green cards above (Anthropic Financial Services / OpenBB / Alpha Vantage / Tiingo MCP) plus TradingAgents in purple form a complete v1 stack — pulp aggregation, multi-provider data wrapping, multi-agent draft critique. Zero license walls for a single-author morning-brief workflow. The eight orange cards are mostly aspirational reference for v2+: every one of them has a pattern worth stealing (matrix UI, bull/bear synthesis, expert-grounded sourcing) without buying the license.

The honest read: we're not data-poor; we're triage-poor. OpenBB + Alpha Vantage + the four major free RSS sources already give Quint more raw signal than he can metabolize in a 5-AM window. The dashboard's job is the cross-ref-against-holdings + voice-conditioned commentary layer — that's where the moat is. Build there, not on the data side.
